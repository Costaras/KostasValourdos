Nashville Housing Data Cleaning

  • View Project Code

Project Summary

This project involves comprehensive data cleaning of the Nashville housing dataset to ensure data integrity and facilitate analysis. The process begins by renaming columns for clarity and ease of manipulation. Dates are formatted consistently by altering column types. Missing values in property addresses are populated using SQL ISNULL and JOIN functions to maintain data completeness. Additionally, addresses are split into individual columns such as Property Address and City, demonstrating proficiency in string manipulation and column updates.

To standardize the dataset, categorical values in the 'SoldAsVacant' column are normalized. Subsequently, duplicates are identified and removed using common table expressions (CTEs) and ROW_NUMBER functions, ensuring unique and accurate data. Unnecessary columns are removed to streamline the dataset. These steps highlight essential data cleaning techniques, making the data ready for subsequent analysis and visualization.
